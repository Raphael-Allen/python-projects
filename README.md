# Python Projects

A collection of Python projects built during self-directed learning, focused on developing practical programming skills through real applications. Each project is built from scratch using object-oriented principles, file persistence, and clean code practices.

---

## Projects

### Expense Tracker
A command-line expense tracking application built using object-oriented Python.

**Features:**
- Add expenses with description, amount, category, and automatic date stamping
- View all expenses in a formatted numbered list
- View total amount spent across all expenses
- Filter expenses by category with a category total
- Delete individual expenses by number
- JSON file persistence — expenses survive between sessions

**Concepts covered:**
- Classes and object-oriented programming
- Dictionaries inside lists as a data structure
- f-strings with float formatting (`:.2f` for currency)
- `datetime` module for automatic date stamping
- JSON file persistence with `json.dump` and `json.load`
- Error handling with `try/except`
- List enumeration, filtering, and accumulator pattern

**Run it:**
```bash
python3 ExpenseTracker.py
```

### To Do List
A command-line task manager with full CRUD functionality and persistent storage.

**Features:**
- Add tasks
- View all tasks in a numbered list
- Mark tasks as complete
- Delete tasks
- JSON file persistence — tasks survive between sessions

**Concepts covered:**
- Classes and object-oriented programming
- List methods and manipulation
- JSON file persistence
- While True runtime loop
- Error handling with `try/except`

**Run it:**
```bash
python3 To_Do_List.py
```
